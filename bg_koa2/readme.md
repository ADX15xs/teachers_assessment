# 目录简介

1. `app`
   1. `controller`： 控制层
   2. `model`： 对象层
   3. `service`： 业务层
2. `bin`
   1. `www`：项目入口，通过它引入`app.js`配置内容
3. `config`： 配置文件（可以配置数据库连接密码）
4. `middelware`： 中间件
5. `public`： 公共文件夹
   1. `images`： 图片
   2. `javascripts`： 页面js逻辑
   3. `stylesheets`： 样式
6. `routes`： 路由（功能为分发请求）
7. `test`： 测试
8. `views`： 视图
9. `app.js`： 配置文件
10. `package.json`： 配置文件
